## 🇬🇧 What is this?

This repository is my **public learning diary**. Every working day, I write a short, focused note about something new I encountered — a SQL query, a Python snippet, a process modeling concept, or a key idea from school (Berufsschule / Blockunterricht).

The goal is simple: **document the learning journey, one commit at a time.**

This is not a tutorial collection. It is not polished. It is honest and incremental — exactly like learning itself.

---

## 🇩🇪 Was ist das hier?

Dieses Repository ist mein **öffentliches Lerntagebuch**. Jeden Arbeitstag schreibe ich eine kurze, fokussierte Notiz über etwas Neues, das ich entdeckt habe — eine SQL-Abfrage, einen Python-Snippet, ein Konzept zur Prozessmodellierung oder eine wichtige Idee aus der Berufsschule (Blockunterricht).

Das Ziel ist einfach: **den Lernweg dokumentieren, einen Commit nach dem anderen.**

Das hier ist keine Tutorial-Sammlung. Es ist nicht perfekt. Es ist ehrlich und schrittweise — genau wie das Lernen selbst.

---

## 📁 Structure — Struktur

The folders grow organically as I progress through my Ausbildung modules.
*Die Ordner wachsen organisch, je weiter ich in meinen Ausbildungsmodulen voranschreite.*

| Folder | Content |
|---|---|
| `sql/` | Queries, joins, database design, optimization |
| `python/` | Scripts, Pandas snippets, automation |
| `bpmn/` | Process modeling diagrams & concepts |
| `etl/` | Data pipeline patterns & integration |
| `school/` | Notes from Berufsschule, organized by Lernfeld |
| `theory/` | Cross-cutting topics: networks, security, project management |

> New folders are added only when a subject is actually covered — no empty placeholders.
> *Neue Ordner werden nur hinzugefügt, wenn ein Thema tatsächlich behandelt wird — keine leeren Platzhalter.*

---

## 📝 Note Template — Vorlage für Notizen

Each note follows this structure / *Jede Notiz folgt dieser Struktur*:

```markdown
# [Title of what I learned today]
*Date: DD-MM-YYYY* | *Category: #python / #sql / #bpmn*

## Context — Kontext
What I was trying to solve, or what was covered in school today.

## Code / Diagram / Formula
Insert the clean snippet, query, or diagram here.

## Key Takeaway — Was ich gelernt habe
2-3 bullet points explaining why this matters and how it works.
```
